◆ isConnected()

bool moza::HidDevice::isConnected ( ) const

Check if the device is in a connected state, useful for determining if the device is disconnected.

Note
Internally, this is determined based on the error code of the last device operation. It is recommended to use this function if a call to the button state retrieval function returns empty, to confirm whether the device has been disconnected
Returns
Returns true if the device is connected; otherwise, returns false

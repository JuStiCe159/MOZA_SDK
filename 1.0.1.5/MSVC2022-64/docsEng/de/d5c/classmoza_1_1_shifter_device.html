◆ getCurrentGear()

int moza::ShifterDevice::getCurrentGear ( ) const

Get the current gear of the shifter.

Returns
The current gear
Return values
-1for reverse gear, 0 for neutral gear, and 1-7 for forward gears. During gear shifting, since the neutral position is passed through, a value of 0 may appear during this process.
Warning
This function waits for the HID report while reading data, until valid data is received or an error occurs. This means the execution time of this function may be relatively long, and it is not recommended to call it in the main thread.
